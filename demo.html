Class Harmony

Helping K-12 schools create balanced class lists

Project Overview

Class Harmony is a comprehensive solution for K-12 schools to create balanced class lists. The application allows school administrators to import student data, collect teacher input through surveys, and utilize optimization algorithms to create balanced classes.

Student Import

Upload student data from CSV or Excel files with a user-friendly interface.

Map columns, review data, and import with confidence.

Class Lists

Create and manage class lists with a drag-and-drop interface.

Automatic class balancing based on various student attributes.

Teacher Surveys

Collect teacher input on student needs and compatibility.

Factor in professional observations for optimal class placement.
